Hallazgo central

Patrón de interacción

"Las respuestas no se centraron en el cambio, sino en entender a qué producto o plataforma aplicaba."

El cliente no procesó el mensaje como una actualización operativa. Lo procesó como una señal sobre el ecosistema de productos — y quiso entender qué lugar ocupa cada uno.

Para el negocio

Implicaciones

Lo que este patrón significa para la operación, el canal y la estrategia de marca.

1

Diferenciación de plataformas no clara

Los clientes no distinguen con claridad entre UnaLanaPay y Peibo como productos diferenciados con funciones distintas.

2

Comunicación genera ambigüedad

Un mensaje operativo de ULP genera ruido de posicionamiento que requiere esfuerzo adicional para aclarar en el canal.

3

Dudas sobre la vigencia de UnaLanaPay

Algunos clientes interpretaron el cambio de horarios como una señal de discontinuidad, preguntando directamente si ULP seguirá operando.

4

Oportunidad de posicionar Peibo

La confusión es también una puerta: el cliente ya está preguntando por Peibo, lo que facilita la conversación de transición.

Insight clave

"Hoy el cliente interpreta el ecosistema como uno solo,
no como productos diferenciados."

Esta percepción no es un error del cliente — es una señal sobre cómo se ha construido la narrativa del ecosistema hasta ahora. Y es el punto de partida para corregirla.
